### ウェストミンスターの鐘  
(英語：Westminster Quarters)  

ウェストミンスター宮殿の時計塔ビッグ・ベンで使われている時鐘のメロディ。  
日本では学校のチャイムとして使われています。

|   Oct4   |  5 |====     |ソ   |
|   Oct4   |  1 |======== |ド   |
|          |  0 |==       |休符 |
|   Oct4   |  4 |====     |ファ |
|   Oct4   |  5 |====     |ソ   |
